Project Overview

Build a web solution utilizing the MEAN stack experience you have that will search for all of the information for a company i.e. type into a web front-end “MSFT” or “Microsoft” and it will then pull back information and sentiment analysis for Microsoft. There is plenty of documentation on-line, specifically the Discovery service which is probably the main IBM Watons API you will use.
